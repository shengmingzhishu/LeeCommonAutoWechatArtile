# 微信公众号文章推送器

一个用于自动发布文章到微信公众号的实用工具，提供友好的交互式命令行界面，让操作更加便捷。

## 🚀 功能特点

- **交互式界面**: 命令行交互，操作简单直观
- **自动浏览器操作**: 自动打开微信公众平台并完成文章发布流程
- **灵活的内容输入**: 支持多行文章内容，自动生成摘要
- **实时状态反馈**: 提供详细的操作进度提示
- **完善错误处理**: 完善的错误处理和用户提示

## 📦 安装方法

### 自动安装（推荐）
```bash
chmod +x install.sh
./install.sh
```

### 手动安装
```bash
pip3 install playwright
playwright install
```

## 🎯 使用方法

### 启动程序
```bash
python3 start.py
```

### 使用流程
1. **启动**: 运行启动脚本
2. **输入文章信息**: 
   - 标题（必填）
   - 作者（可留空，默认"北屿"）
   - 内容（支持多行，输入"END"结束）
   - 摘要（可留空，自动生成）
3. **确认信息**: 程序显示信息供确认
4. **自动推送**: 程序自动打开浏览器完成推送

## 📁 项目文件

- `start.py` - 启动脚本，提供菜单
- `wechat_publisher_cli.py` - 核心功能模块
- `requirements.txt` - 依赖列表
- `install.sh` - 自动安装脚本
- `README.md` - 项目说明

## ⚠️ 使用要求

- Python 3.7+
- 网络连接
- 已登录的微信公众平台账户
- 浏览器可执行环境

## 💡 使用提示

- 内容输入支持多行，"END"结束
- 摘要留空自动生成（前50字符）
- 推送过程会显示详细进度
- 检查网络和登录状态解决错误
