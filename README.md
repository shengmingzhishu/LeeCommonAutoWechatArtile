# 微信公众号文章推送器

一个用于自动发布文章到微信公众号的实用工具，提供友好的用户界面，让操作更加便捷。

## 🚀 功能特点

- **用户友好的界面**: 提供交互式命令行界面，操作简单直观
- **自动浏览器操作**: 自动打开微信公众平台并完成文章发布流程
- **灵活的内容输入**: 支持多行文章内容，自动生成摘要
- **实时状态反馈**: 提供详细的操作进度提示
- **错误处理**: 完善的错误处理和用户提示

## 📦 安装方法

### 方式一：自动安装（推荐）
```bash
chmod +x install.sh
./install.sh
```

### 方式二：手动安装
```bash
# 1. 安装Python依赖
pip3 install -r requirements.txt

# 2. 安装Playwright浏览器驱动
playwright install
```

## 🎯 使用方法

### 快速开始
```bash
python3 start.py
```

### 直接运行
```bash
python3 wechat_publisher_cli.py
```

### 使用流程
1. **启动程序**: 运行启动脚本
2. **输入文章信息**: 
   - 标题（必填）
   - 作者（可留空，默认"北屿"）
   - 内容（支持多行输入，以"END"结束）
   - 摘要（可留空，自动生成）
3. **确认信息**: 程序显示所有输入信息供确认
4. **自动执行**: 程序自动打开浏览器并完成推送

## 📁 文件说明

- `start.py` - 主启动脚本，提供菜单选择
- `wechat_publisher_cli.py` - 交互式命令行版本
- `wechat_publisher_gui.py` - GUI图形界面版本（需要GUI环境）
- `playwright_wechat_push.py` - 原始自动化脚本
- `requirements.txt` - Python依赖列表
- `install.sh` - 自动安装脚本

## ⚠️ 使用要求

### 环境要求
- Python 3.7+
- 网络连接
- 已登录的微信公众平台账户

### 依赖项
- `playwright` - 浏览器自动化库
- `tkinter` - GUI库（仅GUI版本需要）

## 🔧 技术特点

- **异步处理**: 使用Playwright进行浏览器自动化
- **多线程支持**: GUI版本使用多线程避免界面卡顿
- **智能摘要**: 自动从文章内容生成摘要
- **错误恢复**: 完善的异常处理和用户提示

## 💡 使用提示

- 文章内容可以输入多行任意内容
- 摘要留空时自动取内容前50个字符
- 推送过程中会在浏览器中显示操作进度
- 如遇问题请检查网络连接和登录状态

## 🤝 支持

如有问题或建议，请检查：
1. 依赖项是否正确安装
2. 微信公众平台登录状态
